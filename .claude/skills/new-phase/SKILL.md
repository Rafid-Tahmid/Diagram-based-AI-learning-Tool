# new-phase

Starts the next phase of the project.

Steps:
1. Read `CLAUDE.md` and identify the current phase and the next phase
2. Summarize what was built in the current phase (1-2 sentences)
3. Show the next phase's goal and "Done when" criteria
4. Ask: "Ready to start? I'll show a plan before writing any code."
5. On confirmation, produce a bullet-point plan and wait for approval
6. Only begin coding after the plan is approved
